DNS

Это компьютерная система, содержащая соответствие ip адресов и названий сайтов.
Даёт возможность пользователям не запоминать цифровые обозначения сайтов(ip адреса), а вводить удобные текстовые имена сайтов. 
Соответствие этих адресов хранится на dns серверах. 
При запросе какого-либо адреса сайта браузер обращается поочередно к dns серверам, начиная с ближайшего, 
запрос передается к вышестоящим серверам до тех пор пока 6е дойдет до нужного сервера, 
в котором хранится информация о нужном ip адресе  и может быть открыта страница нужного сайта.


Http это стандарт передачи данных в интернете.
Https - это защищённый стандарт передачи данных.
Http передача данных быстрее, 
Https передача данных безопаснее, данные перелаются в зашифрованном виде.

Http/1.1 протокол передает данные в виде текста, 
а его более современная версия http/2 данные сжимает и преобразует в двоичную систему, 
что увеличивает скорость загрузки.
